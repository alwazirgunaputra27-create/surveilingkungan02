## 📋 Alur Penggunaan

1. **Login** - Pengguna memasukkan email dan password
2. **Biodata** - Mengisi data pribadi lengkap dengan validasi
3. **Survey** - Menjawab 5 pertanyaan tentang lingkungan sekolah
4. **Konfirmasi** - Mengunduh file .exe dan menghubungi via WhatsApp

## 🛠️ Teknologi yang Digunakan

- **HTML5** - Struktur website
- **CSS3** - Styling dengan CSS Variables dan Flexbox/Grid
- **Vanilla JavaScript** - Interaktivitas dan validasi form
- **Font Awesome** - Ikon dan UI elements
- **Google Fonts** - Typography
